# Aleron MD Handoff Template

Create one handoff per substantial task. Use a date-prefixed filename, for example:

`2026-05-17-canonical-landing-sync.md`

## Template

# <Task title>

Date:
Agent:
Repo:
Branch:
Base SHA:
Final SHA or patch status:

## Goal

What the task was meant to accomplish.

## Files changed

- `path/to/file`: what changed

## Behavior changed

What a user or reviewer should observe differently.

## Verification

Commands or checks run, with result summary.

## Open risks or follow-up

Anything not verified, intentionally deferred, or needing Jason/Hermes review.
